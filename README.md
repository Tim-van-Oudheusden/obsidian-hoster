# Personal Publishing Website

## Based on
- TypeScript
- Node.js

## Features
- Article Publishing markdown notes written in [Obsidian](https://obsidian.md/)
- Commenting via Giscus
- Frontmatter via [Obsidian Properties](https://help.obsidian.md/properties)
- Graph and backlinks
